IronLog er en treningsapp bygget i React + Vite, deployet på Vercel, med Supabase for auth og sky-lagring.

**Why:** Målet er å publisere appen i App Store via Capacitor når den er ferdig utviklet.

**How to apply:** Prioriter funksjoner som gir verdi for App Store-lansering. Bruk Capacitor for iOS-bygging når klar.

## Status (2026-05-26)
- GitHub: https://github.com/Kjetil1992/ironlog
- Vercel: https://ironlog-phi-liard.vercel.app
- Supabase: https://nmykehymfgjceeczjrsf.supabase.co
- Auth: e-post + passord via Supabase
- Datalagring: sessions og programs i Supabase (JSONB)
